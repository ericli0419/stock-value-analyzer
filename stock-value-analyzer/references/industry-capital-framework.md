# 行业框架与资本配置能力手册（Industry & Capital Allocation Framework）

> 本手册服务于 stock-value-analyzer v2.0 的「定性层」。
> - **行业框架**：把"好行业"从感性判断升级为结构化分析（波特五力 / 生命周期 / 产业链议价 / TAM-SAM-SOM）。
> - **资本配置能力**：评估管理层"赚到的钱用得好不好"——这是巴菲特衡量优秀管理层的第一标准，也是邱国鹭"好公司"长期复利的关键变量。
>
> 本手册不含任何个人持仓、组合联动或交易决策内容，可独立公开使用。

---

## 第一部分：行业结构分析框架

### 1.1 波特五力（Porter's Five Forces）

行业的"先天属性"由五种力量共同决定。每一力越强（对在位企业越不利），行业吸引力越低。

| 力量 | 核心问题 | 对行业评分的影响 |
|---|---|---|
| **现有竞争者竞争强度** | 玩家多不多？打不打价格战？产品同质化吗？ | 越激烈 → 行业格局分越低 |
| **潜在进入者威胁** | 新玩家进来容易吗？有没有牌照/资本/技术/品牌壁垒？ | 壁垒越低 → 进入壁垒分越低 |
| **替代品威胁** | 有没有别的方案能满足同样需求？（如视频会议替代差旅） | 替代品越强 → 需求稳定性分越低 |
| **供应商议价能力** | 上游能不能卡你脖子、随意涨价？ | 供应商越强势 → 定价权/盈利分越低 |
| **买方议价能力** | 下游客户能不能压价、说走就走？ | 买方越强势 → 定价权分越低 |

**评分映射建议**（对应 SKILL.md 模块一行业分析的四维）：
- 现有竞争 + 潜在进入者 → 主要影响「行业格局」与「进入壁垒」两维
- 供应商 + 买方议价 → 主要影响「定价权」维
- 替代品 → 主要影响「需求稳定性」维

**输出要求**：报告行业分析中必须用一句话点评每一力，并指出"哪一力是该行业的决定性力量"。

### 1.2 行业生命周期（Industry Life Cycle）

不同阶段，估值逻辑和风险点完全不同。误判阶段是估值陷阱的重要来源。

| 阶段 | 特征 | 估值逻辑 | 主要风险 |
|---|---|---|---|
| **导入期** | 渗透率 < 5%，玩家少，亏损普遍 | 看 TAM 和渗透斜率，PS / 用户数估值 | 技术路线被颠覆、需求证伪 |
| **成长期** | 渗透率 5%-50%，增速快，开始盈利 | 反向 DCF / PEG / Rule of 40 | 竞争加剧、增速不及预期 |
| **成熟期** | 渗透率 > 50%，增速放缓，格局固化 | DCF / PE-ROE / 自由现金流贴现 | 第二曲线缺失、被动多元化 |
| **衰退期** | 需求萎缩，产能过剩 | 清算价值 / 资产重置 / 高股息 | 价值陷阱（PE 低但持续恶化） |

**关键动作**：
- 明确标注标的所处阶段，并用渗透率/增速/盈利状态三项佐证。
- **警惕"成熟期当成长期估值"**——这是高估值杀的主因（典型：消费电子、互联网平台进入成熟期后仍按成长股给 PE）。
- **警惕"衰退期当价值股抄底"**——PE 低 ≠ 便宜，配合 valuation-traps.md 与 M-Score 检查。

### 1.3 产业链议价地位（Value Chain Position）

同一行业里，处于产业链不同环节的公司命运天差地别。核心问题：**利润在产业链上是怎么分配的？标的卡在哪个环节？**

分析三步：
1. **画出产业链**：上游（原料/设备）→ 中游（制造/组装）→ 下游（品牌/渠道/服务）。
2. **找"微笑曲线"的高点**：通常两端（核心技术/品牌）利润高，中间（代工/组装）利润薄。
3. **判断标的的议价地位**：
   - 是否掌握稀缺资源/核心技术/品牌心智？→ 高议价
   - 是否容易被上游卡脖子或被下游压价？→ 低议价
   - 集中度对比：上游集中度 vs 本环节集中度 vs 下游集中度，谁更分散谁更弱势。

**输出要求**：用一句话回答"这家公司在产业链里是收税的，还是交税的"。

### 1.4 市场空间 TAM / SAM / SOM

成长股估值的天花板由市场空间决定，避免"线性外推增速到无穷"。

| 概念 | 定义 | 用途 |
|---|---|---|
| **TAM**（Total Addressable Market） | 全部潜在市场总规模 | 判断长期天花板 |
| **SAM**（Serviceable Available Market） | 标的产品/区域可触达的细分市场 | 判断现实可争取的盘子 |
| **SOM**（Serviceable Obtainable Market） | 短中期可实际拿下的份额 | 校验增速假设是否合理 |

**反向校验法**：用"反向 DCF 隐含增速 × 持续年限"推算出"N 年后标的的隐含营收/份额"，对照 TAM/SAM——若隐含份额超过行业 TAM 的合理上限（如要求拿下 > 60% 全球市场），则当前估值过度乐观。

## 第二部分：资本配置能力评估（管理层质量的硬核维度）

> **巴菲特原则**：CEO 最重要的工作是资本配置（Capital Allocation）。一家公司每年赚的钱（留存收益 + 折旧 + 融资）有四个去向，配置得当则复利惊人，配置失当则毁灭价值。
> **邱国鹭对应观点**："好公司"的管理层既要会赚钱，更要会花钱——乱铺摊子、追热点并购、高位增发的管理层会持续摧毁股东价值。

### 2.1 现金的四大去向

| 去向 | 健康信号 | 危险信号 |
|---|---|---|
| **① 再投资于主业**（capex / 研发） | 投出去的钱带来 ROIC > WACC 的回报 | 持续高 capex 但 ROIC 走低 → 价值毁灭式扩张 |
| **② 并购扩张**（M&A） | 围绕主业、协同清晰、价格理性、整合成功 | 高溢价跨界并购、商誉激增、频繁"讲故事式"收购 |
| **③ 分红**（dividend） | 派息率稳定且与现金流匹配，可持续 | 借债分红、派息率 > 100%、为撑股价硬分红 |
| **④ 回购**（buyback） | 在**低估时**回购并注销，提升每股价值 | 高位回购、回购仅用于对冲股权激励稀释 |

### 2.2 留存收益再投资回报检验（核心定量锚点）

巴菲特"一美元留存收益至少创造一美元市值"标准的可操作版本：

```
检验法：取过去 N 年（建议 5 年）
  累计留存收益 = 累计净利润 − 累计分红
  市值增量 = 期末市值 − 期初市值（剔除增发/回购影响后更佳）

  若 市值增量 ≥ 累计留存收益 → 管理层把留存的钱用出了 ≥1 倍的价值（✅ 优秀配置）
  若 市值增量 < 累计留存收益 → 留存的钱没有创造对等价值（⚠️ 配置存疑）
```

**配合 time-series-analysis.md 的 ROE 一致性**：高 ROE + 高再投资比例 + 再投资回报维持高位 = 复利机器；高 ROE 但靠分红派光（再投资空间小）= 现金牛但成长有限（估值逻辑不同）。

### 2.3 资本配置评分卡（0-10 分，并入 SKILL.md 公司模块管理层维）

| 评估项 | 优秀（满分） | 一般 | 差 |
|---|---|---|---|
| **再投资回报**（ROIC vs WACC） | ROIC 长期显著 > WACC | 约等于 WACC | ROIC < WACC 仍持续扩张 |
| **并购理性** | 围绕主业、价格审慎、整合成功 | 偶有跨界但可控 | 高溢价跨界、商誉减值频发 |
| **分红/回购理性** | 与现金流匹配、低位回购注销 | 中性 | 借债分红 / 高位回购对冲激励 |
| **留存收益转化** | 5 年市值增量 ≥ 累计留存 | 接近 | 显著低于留存收益 |

### 2.4 股权激励稀释检查（容易被忽略的隐性成本）

- **检查项**：股本是否年年增长？增发是否主要用于股权激励/可转债？回购是否仅用来"填平"激励稀释而非真正减少股本？
- **数据来源**：脚本 `time_series.shares` 序列（股本趋势）+ 现金流量表的"股份回购"科目。
- **红线**：稀释后摊薄 EPS 增速持续低于净利润增速，说明股东被悄悄稀释——需在管理层评分扣分，并在估值时用摊薄股本而非基本股本。

## 第三部分：行业差异化估值路由（与 valuation-models.md 联动）

不同行业不能用同一把估值尺子。本表给出行业 → 估值方法的路由（详细公式见 `valuation-models.md`）。

| 行业类型 | 首选估值法 | 辅助 | 关键陷阱 |
|---|---|---|---|
| **金融**（银行/保险/券商） | PB-ROE、DDM、内含价值（保险） | 拨备覆盖率、不良率调整 | PE 失真；M-Score/Z-Score **不适用**（须确认 applicable=False） |
| **周期**（资源/化工/航运/地产） | 归一化盈利 PE、市值/产能、市值/储量、PB | 跨周期均值 | **周期顶部低 PE = 陷阱**，须用归一化盈利 |
| **成长**（科技/创新药/新消费） | 反向 DCF、PEG、Rule of 40、PS（未盈利） | TAM 反向校验 | 线性外推增速、忽视竞争与渗透天花板 |
| **稳定消费/公用事业** | DCF、PE-ROE、自由现金流贴现、股息贴现 | 历史估值分位 | 把成熟期当成长期估值 |
| **重资产**（制造/基建） | PB、EV/EBITDA、重置成本 | ROIC vs WACC | 折旧政策操纵、capex 黑洞 |

## 第四部分：行业框架在报告中的落地清单

报告「行业分析」与「公司·管理层」模块中，须显式包含：

- [ ] 波特五力逐力一句话点评 + 指出决定性力量
- [ ] 行业生命周期阶段判定（渗透率/增速/盈利三佐证）
- [ ] 产业链议价地位："收税还是交税"一句话定性
- [ ] TAM/SAM/SOM 估算 + 反向 DCF 隐含份额校验（成长股必做）
- [ ] 资本配置评分卡（4 项，0-10 分）并入管理层评分
- [ ] 留存收益再投资回报检验（5 年市值增量 vs 累计留存）
- [ ] 股权激励稀释检查（股本趋势 + 摊薄影响）
